# spotify-hashtag
Um projeto réplica do spotify, feito em full-stack fornecido pela Hashtag Programação. Utilizei node.js, react, mongoDB, javaScript e outras tecnologias. No vídeo, mostro um pouco como funciona o aplicativo.
